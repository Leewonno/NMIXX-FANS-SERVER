# 🎤 K-POP COMMUNITY (Django + GraphQL)

케이팝(K-POP) 팬들을 위한 커뮤니티 플랫폼 개발 프로젝트입니다.

## 📌 기술 스택

- **Backend**: Django 5.2.4, Graphene-Django
- **GraphQL API**: Graphene
- **Database**: PostgreSQL
- **Language**: Python 3.10

## ✨ 주요 기능 (계획 포함)

- 회원 가입 및 로그인 (JWT 인증)
- 아티스트/그룹 게시판
- 게시글 CRUD
- 댓글, 좋아요 기능
- GraphQL 기반 통계 조회 (작성 수, 좋아요 수 등)
